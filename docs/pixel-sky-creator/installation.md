# Installation

Pixel Sky Creator is a Unity asset and can be installed via the Unity Asset Store.


## Requirements

Unity 2021.3.11 or higher.


## Asset structure

```bash
PixelSkyCreator
├── Documentation
│   └── Documentation.pdf
├── Prefabs
│   └── Sky.prefab
├── Scripts
│   ├── MonoBehaviours
│   │   └── Sky.cs
│   ├── ColorBands.cs
│   ├── Dither.cs
│   └── SkyUtils.cs
└── Sprites
    ├── dither1-1px.png
    ├── dither1-2px.png
    ├── dither1-3px.png
    ├── ...
    └── ...
```
